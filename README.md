<div align="center">
  <h2>lent</h2>
  <p>取名为慢，一个vite的mini版本实现，都采用比较慢的方式去完成</p>
  <p>主要用于自己本地的一个开箱即用</p>
  <p>应该只支持windows系统</p>
</div>

---

### 支持

1. 自定义插件
2. 中间件
3. 热更新
4. glob导入
5. 开箱即用的typescript

---

### 获取模板

1. npx lent template (--leetcode or --js --ts)

---

### leetcode模板内置了对于树和链表的数据生成
1. 使用以下接口可以修改port
2. localhost:3000/leetcode?type=tree&demo=[1,2,3]
3. localhost:3000/leetcode?type=listnode&demo=[1,2,3]
